### Proactive Intelligence  “ 見えない未来を観ようとして ”

---

### About はじめに

本研究は Proactive Intelligence をテーマとし、予測など未来生成を意思決定に接続する方法の構築が目的です。<br>
対象は 1D時系列データ、2D視覚表現データ、3D空間データ など多様な観測情報であり、<br>
表現学習および潜在状態モデルを統合することで、状態・兆候・予測・介入評価を一体的に扱い、<br>
より確からしい未来生成を求めます。<br>

本研究では、時間構造解析や潜在ダイナミクスに関する基礎的研究に基づき、<br>
状態・兆候・予測・介入評価を一体的に扱う分析枠組みの構築を行い、<br>
これにより、より確からしい未来生成と意思決定支援の実現を目指します。

特に、製造業、サプライチェーン、社会インフラ、医療分野において、<br>
生成した未来を実運用の改善や信頼性向上へと接続する応用に関心があります。  

これらの研究関心は、製造業・医療分野を中心とした産業領域における実務経験に基づいています。<br>
これまで、3D CAD・3D画像処理・点群データ、ECM・SCMシステムを扱うIT企業にて、<br>
業界向けソリューションの開発および導入支援に従事してきました。特に、データ構築支援やDX推進を通じて、<br>
画像やデータを活用した業務の可視化や意思決定支援など、現場の業務改善に取り組んできました。

現場ではデータ分析では対応できない、将来の状況を見据えた判断や意思決定が求められる場面が多く存在します。<br>
これらの経験から、データのリアクティブな活用にとどまらず、
予測や将来推定を含めた「プロアクティブな情報活用」の重要性を強く認識しました。

この問題意識を背景に、現場で直面する予測・判断の課題を解決するため、画像・データ・AIを統合し、<br>
データから未来を推定し意思決定へ結び付ける手法の研究へと発展させています。<br>
実務経験と研究活動の双方を通じて、産業応用AIの実現と業務改善への貢献を目指しています。

### Collaboration 共創研究のご提案

本研究では、企業様の課題を実データに基づいて検証し、研究成果を学会発表や公開<br>
および実務価値への転換を最終目的とした共創研究のお願いです。（2026年5月以降）

製造業、サプライチェーン、社会インフラ分野を中心に、産業データを用いた共創研究を歓迎します。

- 需要・計画・在庫データ
- センサーデータ（設備）
