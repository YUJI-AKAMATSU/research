## About

本研究は **Proactive Intelligence** をテーマとし、単なる予測精度の向上に留まらず、予測結果を意思決定に接続する方法の構築が目的です。
対象は **1D時系列データ、2D視覚表現、3D空間データ** など多様な観測情報であり、表現学習および潜在状態モデルを統合することで、兆候・予測・介入評価を一体的に扱い、より確からしい未来生成を求めます。
特に、製造業、サプライチェーン、社会インフラといった分野において、生成した未来での実運用の改善や信頼性向上につなげる応用に関心があります。
これらの研究関心は、製造業を中心とした産業分野におけるデータ活用支援やDX推進に関わってきた実務経験に基づきます。
現場で直面する予測・判断の課題を背景に、データから未来を推定し、意思決定に結び付ける方法の研究へと発展しています。
